# CLHS Marks Calculator

A lightweight single-page tool that helps CLHS students compute their weighted average instantly. Each subject score is multiplied by its official weight (total weight = 52) before averaging, mirroring the grading table provided by the school.

## Getting Started

Visit [clhs.bernardkoo.dev](https://clhs.bernardkoo.dev). Enter your marks (0–100) for each subject; blank fields count as 0 until filled. The weighted sum and average update in real time following `(Σ mark × weight) ÷ 52`.
